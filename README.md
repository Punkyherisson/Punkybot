Punkybot
Overview
A Discord bot built with Python and discord.py.

Project Structure
Punkybot.py - Main bot file with command handlers
requirements.txt - Python dependencies
Setup
The bot requires a BOT_TOKEN environment variable to be set with a valid Discord bot token.

Running
Run with: python Punkybot.py

Commands
!ping - Returns "Pong!"
